When to use each strategy
Reference Counting Swift ARC, C++ shared_ptr, Python CPython. Immediate reclaim, zero pause. Fails on cycles unless augmented with a cycle detector.
Mark & Sweep Ruby MRI, OCaml, early JVMs. Handles cycles. Stop-the-world pause during collection proportional to live set size.
Generational JVM G1/ZGC, .NET CLR, V8, Go. Best throughput for typical workloads. Minor GC stays fast; major GC is rare.
